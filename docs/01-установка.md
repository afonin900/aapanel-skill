# Установка aaPanel Skill

## Требования

- **Claude Code** установлен и работает
- Доступ к серверу с aaPanel
- API Key aaPanel настроен в панели
- IP вашей машины в whitelist панели

## Шаг 1: Клонировать репозиторий

```bash
git clone https://github.com/professordyx/aapanel-skill.git
cd aapanel-skill
```

## Шаг 2: Глобальная установка команды `aapanel`

```bash
bash install.sh
```

Скрипт установит `aapanel` в `/usr/local/bin` (с sudo) или `~/.local/bin` (без sudo).

## Шаг 3: Добавить сервер

```bash
aapanel servers add hetzner https://YOUR_SERVER_IP:17198 YOUR_API_KEY default
```

Конфиг серверов хранится в `~/.aapanel/servers.conf`.

## Шаг 4: Настроить whitelist IP

1. Откройте панель aaPanel в браузере
2. Перейдите в **Settings > API Interface**
3. Активируйте API если не активирован
4. Добавьте IP вашей машины в **IP Whitelist**

```bash
# Узнать свой IP
curl -s https://api.ipify.org
```

## Шаг 5: Проверить подключение

```bash
aapanel system GetSystemTotal
```

Если вернулся JSON с информацией о системе — всё работает.

## Шаг 6: Установить skill в Claude Code

```bash
mkdir -p ~/.claude/skills/aapanel
cp SKILL.md ~/.claude/skills/aapanel/
cp -r scripts ~/.claude/skills/aapanel/
cp -r references ~/.claude/skills/aapanel/
```

## Структура после установки

```
~/.aapanel/
└── servers.conf          # Конфиг серверов (создаётся через 'aapanel servers add')

~/.claude/skills/aapanel/
├── SKILL.md              # Определение skill
├── scripts/
│   └── aapanel_api.sh    # API скрипт
└── references/
    ├── api-catalog.md     # 170+ endpoints
    └── supabase-react-setup.md
```

## Управление несколькими серверами

```bash
# Добавить серверы
aapanel servers add hetzner https://159.69.216.152:17198 API_KEY_1 default
aapanel servers add backup  https://10.0.0.2:17198       API_KEY_2

# Список серверов (* = default)
aapanel servers list

# Вызов на конкретном сервере
aapanel --server backup system GetSystemTotal

# Сменить сервер по умолчанию
aapanel servers default backup

# Удалить сервер
aapanel servers remove backup
```
