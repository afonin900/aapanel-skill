# Быстрый старт — aaPanel API

Практические примеры использования команды `aapanel`.

## Синтаксис

```bash
aapanel [--server <имя>] <категория> <действие> ['<params_json>']
```

## Доступные категории

`system` `ajax` `site` `files` `database` `ftp` `firewall` `crontab` `plugin` `ssl` `config` `data` `nodejs` `python` `proxy` `safe` `safe_ssh` `logs` `server`

---

## Система

```bash
# Общий статус системы
aapanel system GetSystemTotal

# Информация о дисках
aapanel system GetDiskInfo

# Метрики в реальном времени
aapanel system GetNetWork

# Перезапустить Nginx
aapanel system ServiceAdmin '{"name":"nginx","type":"restart"}'

# Освободить память
aapanel system ReMemory
```

## Файлы

```bash
# Список файлов в директории
aapanel files GetDir '{"path":"/www/wwwroot"}'

# Создать директорию
aapanel files CreateDir '{"path":"/www/wwwroot/my-app"}'

# Создать файл
aapanel files CreateFile '{"path":"/www/wwwroot/my-app/index.html"}'

# Прочитать содержимое файла
aapanel files GetFileBody '{"path":"/www/wwwroot/my-app/index.html"}'

# Сохранить содержимое
aapanel files SaveFileBody '{"path":"/www/wwwroot/my-app/index.html","data":"<h1>Hello World</h1>","encoding":"utf-8"}'

# Скачать файл по URL
aapanel files DownloadFile '{"url":"https://example.com/file.zip","path":"/www/wwwroot","filename":"file.zip"}'

# Распаковать ZIP
aapanel files UnZip '{"sfile":"/www/wwwroot/file.zip","dfile":"/www/wwwroot/my-app","type":"zip"}'

# Запаковать
aapanel files Zip '{"sfile":"/www/wwwroot/my-app","dfile":"/www/wwwroot/my-app.zip","type":"zip"}'

# Права доступа
aapanel files SetFileAccess '{"filename":"/www/wwwroot/my-app","user":"www","access":"755","all":true}'

# Удалить файл / директорию
aapanel files DeleteFile '{"path":"/www/wwwroot/file.zip"}'
aapanel files DeleteDir '{"path":"/www/wwwroot/old-app"}'
```

## Сайты

```bash
# Список сайтов
aapanel data getData '{"table":"sites","limit":20,"p":1,"type":-1}'

# Создать сайт
aapanel site AddSite '{"webname":"{\"domain\":\"mysite.com\",\"domainlist\":[],\"count\":0}","path":"/www/wwwroot/mysite.com","type_id":0,"type":"PHP","version":"00","port":"80","ps":"My Site"}'

# Остановить / запустить сайт
aapanel site SiteStop '{"id":1,"name":"mysite.com"}'
aapanel site SiteStart '{"id":1,"name":"mysite.com"}'

# Удалить сайт (с файлами и базой)
aapanel site DeleteSite '{"id":1,"webname":"mysite.com","ftp":1,"database":1,"path":1}'
```

## Базы данных MySQL

```bash
# Список баз данных
aapanel data getData '{"table":"databases","limit":20,"p":1}'

# Создать базу данных
aapanel database AddDatabase '{"name":"mydb","db_user":"myuser","password":"StrongPass123!","codeing":"utf8mb4","address":"127.0.0.1","dtype":"MySQL","ps":"App DB"}'

# Сменить пароль
aapanel database ResDatabasePassword '{"id":1,"name":"mydb","password":"NewPass456!"}'

# Разрешить удалённый доступ
aapanel database SetDatabaseAccess '{"name":"mydb","access":"%"}'

# Резервная копия
aapanel database ToBackup '{"id":1}'

# Импорт SQL
aapanel database InputSql '{"file":"/www/backup/dump.sql","name":"mydb"}'
```

## Firewall

```bash
# Список правил
aapanel firewall GetList '{"p":1,"limit":50}'

# Открыть порт
aapanel firewall AddAcceptPort '{"port":"3000","type":"tcp","ps":"React App"}'

# Открыть диапазон портов
aapanel firewall AddAcceptPort '{"port":"8000-8100","type":"tcp","ps":"Services"}'

# Заблокировать IP
aapanel firewall AddDropAddress '{"port":"1.2.3.4","type":"drop","ps":"Attacker"}'
```

## Node.js

```bash
# Список проектов
aapanel nodejs get_project_list

# Создать проект
aapanel nodejs create_project '{"project_name":"my-react","project_path":"/www/wwwroot/my-react","run_script":"npm start","node_version":"20","port":"3000"}'

# Установить зависимости
aapanel nodejs install_packages '{"project_name":"my-react"}'

# Запустить / остановить / перезапустить
aapanel nodejs start_project '{"project_name":"my-react"}'
aapanel nodejs stop_project '{"project_name":"my-react"}'
aapanel nodejs restart_project '{"project_name":"my-react"}'

# Reverse proxy (Nginx → Node)
aapanel nodejs bind_extranet '{"project_name":"my-react"}'

# Логи
aapanel nodejs get_project_log '{"project_name":"my-react"}'
```

## SSL/HTTPS

```bash
# Запросить сертификат Let's Encrypt
aapanel ssl apply_cert_api '{"domains":["mysite.com","www.mysite.com"],"auth_type":"http"}'

# Проверить SSL сайта
aapanel site GetSSL '{"siteName":"mysite.com"}'
```

## Cron Jobs

```bash
# Список задач
aapanel crontab GetCrontab

# Создать задачу (запускать скрипт каждый день в 3:00)
aapanel crontab AddCrontab '{"name":"Daily Backup","type":"day","hour":"3","minute":"0","sBody":"bash /scripts/backup.sh","sType":"toShell"}'

# Запустить немедленно
aapanel crontab StartTask '{"id":1}'
```

## Программы

```bash
# Список установленного ПО
aapanel plugin get_soft_list

# Установить Node.js
aapanel plugin install_plugin '{"sName":"nodejs","version":"20"}'

# Установить Docker
aapanel plugin install_plugin '{"sName":"docker","version":"latest"}'
```

## Работа с несколькими серверами

```bash
# На конкретном сервере
aapanel --server hetzner system GetSystemTotal
aapanel --server prod nodejs get_project_list

# На сервере по умолчанию (без флага)
aapanel system GetSystemTotal
```
