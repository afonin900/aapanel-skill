# Аутентификация API aaPanel

## Как работает

API aaPanel использует систему аутентификации на основе **подписи по timestamp**. Каждый запрос должен содержать два специальных параметра.

## Параметры аутентификации

| Параметр | Описание | Как генерировать |
|----------|----------|-----------------|
| `request_time` | Текущий Unix timestamp | `date +%s` |
| `request_token` | MD5 хэш для верификации | `md5(str(timestamp) + md5(api_key))` |

## Алгоритм

```
1. Получить текущий timestamp (Unix epoch в секундах)
2. Вычислить MD5 от API Key
3. Конкатенировать: str(timestamp) + md5(api_key)
4. Вычислить MD5 от результата
5. Передать оба параметра как POST данные
```

## Пример на Bash

```bash
API_KEY="YOUR_API_KEY"
SERVER_URL="https://YOUR_SERVER_IP:17198"
TIMESTAMP=$(date +%s)

KEY_MD5=$(echo -n "$API_KEY" | md5sum | awk '{print $1}')
TOKEN=$(echo -n "${TIMESTAMP}${KEY_MD5}" | md5sum | awk '{print $1}')

curl -sk -X POST "${SERVER_URL}/system?action=GetSystemTotal" \
  -d "request_time=${TIMESTAMP}&request_token=${TOKEN}"
```

## Пример на Python

```python
import hashlib
import time
import requests

API_KEY = "YOUR_API_KEY"
BASE_URL = "https://YOUR_SERVER_IP:17198"

def get_auth():
    timestamp = str(int(time.time()))
    key_md5 = hashlib.md5(API_KEY.encode()).hexdigest()
    token = hashlib.md5((timestamp + key_md5).encode()).hexdigest()
    return {"request_time": timestamp, "request_token": token}

response = requests.post(
    f"{BASE_URL}/system?action=GetSystemTotal",
    data=get_auth(),
    verify=False  # self-signed SSL
)
print(response.json())
```

## Пример на TypeScript/Node.js

```typescript
import crypto from 'crypto';

const API_KEY = process.env.AAPANEL_API_KEY!;
const BASE_URL = process.env.AAPANEL_URL!;

function getAuth() {
  const timestamp = Math.floor(Date.now() / 1000).toString();
  const keyMd5 = crypto.createHash('md5').update(API_KEY).digest('hex');
  const token = crypto.createHash('md5').update(timestamp + keyMd5).digest('hex');
  return { request_time: timestamp, request_token: token };
}

async function apiCall(endpoint: string, params: Record<string, string> = {}) {
  const auth = getAuth();
  const body = new URLSearchParams({ ...auth, ...params });

  const response = await fetch(`${BASE_URL}${endpoint}`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
    body: body.toString(),
  });
  return response.json();
}
```

## Важные моменты

1. **IP Whitelist**: IP вызывающей машины ДОЛЖЕН быть в whitelist панели
2. **HTTPS**: Сервер использует self-signed сертификат — отключите проверку SSL (`-k` в curl)
3. **Content-Type**: Всегда `application/x-www-form-urlencoded`
4. **Метод**: Всегда `POST` (никогда GET)
5. **Timeout**: Рекомендуется 30 секунд для обычных операций

## Хранение credentials

API Key и URL сервера хранятся в `~/.aapanel/servers.conf`. Никогда не хардкодьте credentials в коде или конфигах репозитория.

```bash
# Добавить сервер в конфиг
aapanel servers add myserver https://YOUR_IP:17198 YOUR_API_KEY default

# Просмотр (ключи не отображаются)
aapanel servers list
```
